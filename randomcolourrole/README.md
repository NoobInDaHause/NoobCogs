# RandomColourRole Help

Have a role that changes colour every 10 minutes.<br/><br/>May or may not be API intense but the cog is cool.<br/>Due to API rate limits you can only have 1 randomcolourrole pre guild.<br/>The role colour changes every 20 minutes or so depending on how many guilds the bot is in.

# randomcolourroleset
 - Usage: `[p]randomcolourroleset `
 - Aliases: `rrset`
 - Checks: `guild_only`

Settings for the RandomColourRole.

## randomcolourroleset status
 - Usage: `[p]randomcolourroleset status <state> `

Toggle whether to enable or disable the RandomColourRole.

## randomcolourroleset resetcog
 - Usage: `[p]randomcolourroleset resetcog `
 - Restricted to: `BOT_OWNER`

Reset the RandomColourRole cogs whole config.<br/>Bot owners only.

## randomcolourroleset showsettings
 - Usage: `[p]randomcolourroleset showsettings `
 - Aliases: `ss`

See the current guild settings for the RandomColourRole.

## randomcolourroleset role
 - Usage: `[p]randomcolourroleset role <role> `

Set the guilds rainbow role.

## randomcolourroleset reset
 - Usage: `[p]randomcolourroleset reset `

Reset the RandomColourRoles guild settings.
